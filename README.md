## Estrutura do Projeto

- **Dados**: Contém os arquivos de dados utilizados para a análise.
- **Relatórios**: Contém os arquivos .pbix do Power BI.
- **Documentação**: Contém a documentação do projeto, incluindo este README e quaisquer outros documentos relevantes.

## Pré-requisitos

- **Power BI Desktop**: Para visualizar e editar os arquivos .pbix.
- **Conta Power BI**: Para publicar relatórios no serviço Power BI, se necessário.

## Configuração do Ambiente

1. **Instale o Power BI Desktop**:
   - Baixe e instale a versão mais recente do Power BI Desktop [aqui](https://powerbi.microsoft.com/desktop/).

2. **Clone este repositório**:
   ```bash
   git clone https://github.com/wvdomingos/PowerBI
   cd PowerBI
   ```

## Importação dos Dados

Os dados estão localizados na pasta `Dados`. Você pode importar esses arquivos diretamente no Power BI Desktop. Siga os passos abaixo:

1. Abra o Power BI Desktop.
2. Clique em **Obter Dados**.
3. Selecione a fonte de dados apropriada (CSV, Excel, etc.).
4. Navegue até a pasta `Dados` e selecione o arquivo desejado.
5. Carregue os dados no Power BI.

## Construção dos Dashboards

Os arquivos .pbix na pasta `Relatórios` contêm dashboards pré-construídos. Para visualizar ou editar esses dashboards:

1. Abra o Power BI Desktop.
2. Clique em **Arquivo** > **Abrir**.
3. Navegue até a pasta `Relatórios` e selecione o arquivo .pbix.
